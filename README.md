# Hyper-Yoncé

A Queen Bey inspired theme for [Hyper](https://hyper.is/).  Based on the [Yonce Theme](https://yoncetheme.com/).

![screenshot.png](screenshot.png)

## Install

Add `hyper-yonce` to the plugins list in your `~/.hyper.js` config file.
